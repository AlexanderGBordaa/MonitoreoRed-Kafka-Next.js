# TP7 - Monitor de Nodos (reconstruido)

Proyecto reconstruido automáticamente.

Backend
- Ejecutar en PowerShell:

```powershell
$env:PORT = '4000'; npm install; npm start
```

Frontend
- Abrir otra terminal y desde la raíz ejecutar:

```powershell
npm install --prefix frontend
npm run dev --prefix frontend
```

Notas
- Si usas Windows y OneDrive, verifica permisos de la carpeta.
- Si hay errores de dependencias, corre `npm install` en `frontend` y la raíz según sea necesario.
